CLI Reference¶

usage: python3 mc2.py [-h] {upload,run,download} ...

Positional Arguments¶

command

Possible choices: upload, run, download

Command to run.

Sub-commands:¶

upload¶

Encrypt and upload data.

python3 mc2.py upload [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Encrypt data in Opaque SQL format

Default: False

--xgb

Encrypt data in Opaque XGBoost format

Default: False

run¶

Attest the MC2 deployment and run your script.

python3 mc2.py run [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Run the Opaque SQL code specified in the config

Default: False

--xgb

Run the Opaque XGBoost code specified in the config

Default: False

download¶

Download and decrypt results from your computation.

python3 mc2.py download [-h] [--sql] [--xgb]

Named Arguments¶

--sql

Decrypt data in Opaque SQL format

Default: False

--xgb

Decrypt data in Opaque XGBoost format

Default: False
